# Literature Prize Winners - Java 8

## Overview

This project is a **Java 8 console-based application** that manages and processes data about **Literature Prize Winners**. It was developed as a part of a programming coursewok to demostrate how structured data can be modelled, stored and queried using **object-oriented programming principles**.

The system loads and represents information about laureate prize winners and allows users to **search, filter, and display records** based on specific criteria.

## Key Purpose

The purpose of this project is to demonstrate: 

- how real-world data (laureate winners) can be represented using Java classes
- how to organise and manipulate collections of objects
- how to implement **searching and filtering logic**
- how to build a **data-driven console application**

It focuses on appluing **OOP concepts**, rather than building a graphical system.

## Problem Explained

Large datasets, such as list of prize winners, require structured handling to:

- store records efficiently
- retrive specific information
- filter results based on conditions
- present data in a readable format

This project simulates a simplified system where **laurate data is processed and queried** through a command-line interface.

## Project Execution

The application performs the following operations:

- Loads and stores **laureate winners data**
- Displays list of winners
- Allows **searching/filtering** based on:
    - year
    - category
    - name or other attributes
- Outputs results to the console in a structured format

The program demonstrates how datasets can be **processed and quired programmatically**.

## How the System Works

1. The program starts and loads the dataset
2. Laureate records are stored in memory
3. The user is prompted for input (search/filter)
4. The system processes the request
5. Matching results are displayed
